keyboard shortcuts

Add new keyboard shortcuts by setting a url pattern, key sequence, and a sequence of operations. Url patterns follow js regex (e.g. ^https?://(www\.)?google\.com.*). Key sequences consist of short name of key delimited by + (e.g. alt+shift+s). Actions are a sequence of operations to accomplish an action. See Wiki for additional details. Expand the action textarea to see the entire action
